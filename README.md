MySQL Layoffs Data Cleaning Project

Overview

Cleaned a messy layoffs dataset using MySQL to improve data quality and prepare it for analysis.
Problems Found
- Duplicate rows
- Null values
- Inconsistent industry names
- Country formatting issues
- Text-based date fields

Cleaning Steps
- Created staging tables
- Removed duplicates using ROW_NUMBER()
- Standardized text fields
- Converted dates to DATE format
- Handled null values
- Filled missing industry data
Skills Used
MySQL, Window Functions, Joins, Data Cleaning

Result
Produced an analysis-ready dataset suitable for dashboards and reporting.
